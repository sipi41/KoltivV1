Steps to run the sample:

1) Download .Net 10 SDK and install it. https://dotnet.microsoft.com/en-us/download/dotnet/10.0
2) Install the latest (v10) EF core tools by running the following command: dotnet tool install --global dotnet-ef
3) using a CMD, get into the folder where all the file are (where Program.cs is) and Run the command: dotnet-ef database update. This will create the database and seed with initial parameters.
4) Run the app and enjoy!
